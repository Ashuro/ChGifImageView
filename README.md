## ChGifImageView

<p align="center">
<img src="./assets/ChGifImageViewLogo.png" width="300" height="250" />
</p>

支持通过URL、图片播放Gif，也支持直接读取Gif文件。对内存做了一定程度的优化。

###### 如何使用

```ruby
pod 'ChGifImageView'
```
